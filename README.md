# CSS Workshop

Interactive workshop covering CSS basics, CSS preprocessors (Sass), and Tailwind CSS - focusing on maintainable and robust styles.
The three tasks are explained in each subdirectory's README:

1. [Basic CSS](./task1-css/README.md)
2. [Sass](./task2-sass/README.md)
3. [Tailwind CSS](./task3-tailwind/README.md)

[Snippets](./snippets) contains some code examples for each part of the workshop that are used in the presentation.
